Background property

background-color

background-color - це властивість в каскадних таблицях стилів (CSS), яка використовується для визначення кольору фону елемента веб-сторінки. Властивість вказується у вигляді ключового слова, наприклад, назви кольору (наприклад, "red", "blue", "green") або шістнадцяткового коду кольору (наприклад, "#FF0000" для червоного кольору). Властивість background-color може також приймати значення "transparent", яке робить фон елемента прозорим, тобто дозволяє підсвічувати задній фон, якщо він існує.

background-image

background-image - це CSS властивість, яка визначає зображення, яке буде використовуватися як фон для елемента веб-сторінки. Зображення може бути вказане як шлях до файлу зображення або урл (URL) зображення в Інтернеті.
